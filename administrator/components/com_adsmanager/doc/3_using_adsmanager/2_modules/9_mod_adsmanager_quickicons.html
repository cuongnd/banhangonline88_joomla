Mod_adsmanager_quickicons

This module is available only with the gold pack of AdsManager. It will display a list of icons on the home page of the Joomla administrator.

These icons will allow you to go quickly to the page of the component you want.

The design isn't the same for the version 2.5 and 3.0 of Joomla.

You can choose which icons you want in the module options.

Mod_adsmanager_advancedsearch

This module works exactly like the mod_adsmanager_search module. Except, when the user will click on the 'Advanced Search' link, specific fields will be displayed dynamically in the module.

To define the fields that will be displayed in the search, you need to go to the 'Advanced Search' link in the AdsManager Submenu.

Drag and drop the fields you want into the corresponding column.
If you don't set any field in the Advanced Search column, the link will not appear in the search form.

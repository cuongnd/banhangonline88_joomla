Plugins

Introduction

In the free version of AdsManager, 6 plugins are installed automatically.

  • AdsManagerContent - Captcha
  • AdsManagerContent - ReCaptcha
  • AdsManagerContent - Social

In the other versions, plugins for AdsManager are installed automatically along with the PaidSystem component.

  • Invoicing Adsmanager Generator Plugin
  • sh404sef - Adsmanager support plugin
  • sh404sef - PaidSystem support plugin

This article may not contains all the settings available in the configuration of AdsManager. New settings are added regularly.

General

The general tab contains various settings regarding AdsManager.

Download ID
This download ID will allow you to update AdsManager from the Joomla interface.

Email Admin
The admin notifications will be sent to that address.

Name Admin
The name of the admin.

Automatically Published
If set to No, the administrator will need to validate each ads before their publications.

Cron Type
How background tasks like ad expiration should be executed.

Activate Ad Preview before publication
Show the preview button in the ad edit page.

Validation on update
The administrator will need to re-validate the ads after their modifications.

Publication date
Will allow the user to choose the publication date of the ad.

Submission Type
You can choose which types of users can post ads.

Allow ads at root level
Forbid to post ads in parents categories.

Number of Categories
Number of categories in which an ad can be posted.

Number of Ads by User
Maximum number of published ads by users. You can add more fields to limit the number of ads by usergroups

Metadata
You can select how to display the metadatas.

Display RSS Link
Will display the RSS link.

Display print button
Will display the print button.

Show accept rules checkbox
Will display the accept rules checkbox.

Redirect after saving
You can choose on which page the user will be redirected after creating the ad.

Field for global filter Module
Name of the field to use for global filtering.
The global filtering feature will use the searched keyword on all the pages of AdsManager.

Global filtering on user ads view
Apply global filtering on user ads view.

Categories

Category display type
Allow you to choose how to display the category list

Display the categories list label
Show or hide the category list label

Display

Edit Form - Wizard
Display the edit form as a wizard (multiple steps)

Load Juloa Bootstrap2
Load the Bootstrap 2 style on your website, set it to No if your template uses Bootstrap 2 or later.

Ads per page
Number of ads per page displayed in the list view.

Display Mode
Choose how the ads will be displayed in the list view.

Display Date in date column
Show or hide the date in the date column in table mode in the list view.

Display the username in date column
Show or hide the username in the date column in table mode in the list view.

Display the number of view in date column
Show or hide the number of view in the date column in table mode in the list view.

Display Date column
Show or hide the date column in table mode in the list view.

Number of categories per row
Number of category displayed in a row in the front view.

Display Ads counts per Category
Show or hide the number of ads per category in the front view.

Latest ads
Show or hide the latest ads block in the front view

Latest Ads Col Count
The number of cols used when the latest ads are displayed. (Set by default to 3 to have 3 ads in a row).

Latest Ads Rows Count
Number of rows displayed by the latest ads block. (Set to 1 by default)

Display Top Menu
Show or hide the general menu on all views of AdsManager

Display Sort Option
Show or hide the sort field at the top of the ads list in list view.

Display Search TextField
Show or hide the search textfield at the top of the ads list n list view.

Display Pathway
Show or Hide the breadcrumbs of AdsManager.

Show New Indicator
Show or hide the new icon in list view.

Number Days in new
Choose how many days an ad will be seen as new on your website.

Show HOT Indicator
Show or hide the hot icon in list view.

Number Hits for hot
Choose how many views an ad must have to be seen as hot on your website.

Display category on ads List
Show or hide the category label beside the title of the ads

The submenu

Configuration: Displays the AdsMAnager configuration form.
Fields: Displays the fields list.
List View: Displays how the fields will be organized in the ads list.
Details View: Displays how the fields will be organized in the ads detail.
Categories: Displays the categories list.
Ads: Displays the ads list.
Plugins: Displays the adsManager plugins list.
Field Images: Displays the available images for the image type fields.
Tools: Displays the available tools for AdsManager
